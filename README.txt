# Aaron's Piano Exam Buddy v4.2

## Version 4.2 UI update
- Reduced the home screen title card height.
- Reduced the home screen logo/title scale so setup fits better at 100% browser zoom.
- Rebuilt the game screen with fixed sections:
  - compact top bar
  - timer area
  - fixed question/image area
  - fixed answer button area at the bottom
- Added a fixed image window. When there is no picture, the space stays empty but still keeps the layout stable.
- Kept Version 4.1 question banks and Version 4.0 Challenge Mode logic.

## How to run
Unzip the package and open index.html in a browser.
